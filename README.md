# YoroTelecom Billing System

Another pet project to refresh my knowledge of java. A Java-based telecommunications billing system that manages subscribers, subscriptions, and billing operations. It was fun to work on it - I even had some new features in mind to be added. "Nothing too fancy, but still very solid". README was autogenerated to save time.

## Features

- **Subscriber Management**: Create and manage customer accounts with balance tracking
- **Subscription Handling**: Support for multiple subscription types (Music, Internet, TV)
- **Billing Operations**: Subscribe/unsubscribe to services, credit/debit accounts
- **Discount System**: Automatic 15% discount when customers have all three subscription types
- **Operation History**: Complete audit trail of all billing operations
- **Error Handling**: Custom exceptions for various business logic scenarios

## Key Components

- `Subscriber`: Customer record with ID, name, balance, and active subscriptions
- `Subscription`: Service offerings with ID, name, price, and type
- `BillingService`: Core business logic for subscription management and billing
- Repository pattern for data access (`SubscriberRepository`, `SubscriptionRepository`)
- Custom exception classes for business rule validation

## Requiriments

Java JDK  (version 8 or later)

## Sample Output

The program demonstrates:
- Creating subscribers and subscriptions
- Applying bundle discounts (15% off when having all three services)
- Processing account credits and debits
- Tracking operation history
- Error handling for insufficient funds and missing subscriptions
